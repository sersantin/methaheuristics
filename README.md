## Travelling Salesman Problem
 
 Research about metaheuristics applied to the problem of the Travelling Salesman.
 
 ### Methaheuritics
 * Hill-Climbing
 * Simulated-Annealing
 * Variable Neighborhood Search 
 * Taboo Search
